# Quick Start Guide - Media Basher

## Immediate Local Testing

The install.sh script is located at `/app/install.sh` and ready to use!

```bash
# Make it executable
chmod +x /app/install.sh

# Run the installer
sudo /app/install.sh
```

## For Distribution via GitHub

Once you push to GitHub, users will be able to install with:

```bash
wget https://raw.githubusercontent.com/OfficialMikeJ/MEDIA-BASHER/main/install.sh
sudo bash install.sh
```
